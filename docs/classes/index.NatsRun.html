Class NatsRun

NatsRun is a router that matches NATS subject patterns to handlers. It provides Express/Koa-like routing capabilities for NATS messages, allowing you to define handlers for specific subject patterns.

Example

const router = new NatsRun();

// Add a handler for user creation
router.add('user.created', async (msg) => {
  console.log('New user:', msg);
});

// Handle a message
await router.handle('user.created', { id: 1, name: 'John' });

Methods

add

  • add(pattern, handle): void

  • Adds a handler for a specific subject pattern

    Parameters

    • pattern: string

      The NATS subject pattern to match

    • handle: Handler | Handler[]

      Handler function(s) to execute when pattern matches

    Returns void

    Example

    // Single handler
    router.add('user.created', async (msg) => {
      console.log('New user:', msg);
    });

    // Multiple handlers
    router.add('user.updated', [
      async (msg) => { console.log('Handler 1:', msg); },
      async (msg) => { console.log('Handler 2:', msg); }
    ]);

  • match(subject?): Handler[]

  • Returns all handlers that match the given subject

    Parameters

    • subject: string = ''

      The NATS subject to match

    Returns Handler[]

    Array of matching handlers, sorted according to the configured strategy

    Example

    const handlers = router.match('user.123.updated');
    // Returns handlers matching 'user.123.updated', 'user.*.updated', etc.
